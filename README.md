# Probabilistički modeli u digitalnoj obradi slike
Ovaj repozitorij sadrži implementaciju algoritama obrađenih u okviru seminarskog rada iz predmeta Signali i sistemi II.

## Sadržaj
* `Sumovi.ipynb` - Implementacija modela šuma (Gauss, S&P, Speckle) i filtara (Medijan, Adaptivni Wiener, Lee).
* `GMM_Segmentacija.ipynb` - Segmentacija slike i detekcija oštećenja pomoću Modela Gaussovih mješavina.
